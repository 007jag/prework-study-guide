HTML

  • The head element contains informatiom about the webpage.
  • The body element represents the visible content shown to the user.
  • An element is a type of content on a webpage. (Text elements, Image lements, Header elements, List elements)
  • elemnts are also used to divide content on page, and also to describe content to browsers. common elemnt types used to devide and describe a page are meta-related, content division, and semantic elemnts
  • HTML elements work together to creat the structure of a webpage, and provide framework
  • The (head) element contains information, or metadata, that helps the browser to render the page correctly. Elements included within the (head) are not visible when a user looks at a webpage.
  • The (body) element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element.
  • (!DOCTYPE) tells the browser what document type it should expect when its opened
  • (html) opening tag, which represents the root, or start, of our HTML document
  • inside the html tag is lang="en" which tells th ebrowser what language your page is in
  • An HTML attribute defines additional information about an element.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users.
  • The (style) element defines the CSS styles associated with a websit
  • The (base) element defines the base URL for a webpage.
  • The (link) element connects an external resource to the HTML document.
  • The (meta) element defines metadata such as the character set, description, keywords, author, and viewport.
  • The (script) element embeds client-side scripts in the HTML document.
  • The (head) element in this case defines the following three elements: The charset element sets our character encoding. The http-equiv element sets an HTTP header for content. The viewport element sets the default view for our webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
